Tutorial: Using mysql.connector in Python

Introduction

The mysql.connector library is used to connect Python to a MySQL database. In this tutorial, we will show you how to install the library, connect to a database, and perform basic operations.

Installation

To install the mysql.connector library, you can use pip:

pip install mysql-connector-python

Connecting to the Database

To connect to a MySQL database, you will need the database credentials. Here is an example of how to make the connection:

import mysql.connector

connection = mysql.connector.connect(
    host="localhost",
    user="your_username",
    password="your_password",
    database="your_database"
)

if connection.is_connected():
    print("Connected to the database")

Basic Operations

Below are some basic operations you can perform with mysql.connector:

1. Creating a Table

cursor = connection.cursor()
cursor.execute("CREATE TABLE example (id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(255), age INT)")

2. Inserting Data

sql = "INSERT INTO example (name, age) VALUES (%s, %s)"
val = ("John", 28)
cursor.execute(sql, val)
connection.commit()
print(cursor.rowcount, "record(s) inserted.")

3. Selecting Data

cursor.execute("SELECT * FROM example")
results = cursor.fetchall()
for row in results:
    print(row)
