Tutorial: Basic HTML and CSS

Structuring a Basic HTML File

To create a basic HTML file, you need the following structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Title Here</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

This basic structure includes the document type declaration, the HTML tag with a language attribute, a head section with meta tags and a title, and a body section containing your content.

Showing Images and Links

To display images and links, you can use the following HTML tags:

Displaying an Image

Use the <img> tag to display an image. Here’s an example:

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

This code will display an image from the provided URL. Always include the alt attribute to describe the image.

Creating a Link

Use the <a> tag to create a link. Here’s an example:

<a href="https://www.example.com">Visit Example</a>

This code creates a clickable link that takes the user to the specified URL.

Styling a Website with CSS

You can style your website using CSS. Here’s an example of how to include CSS in your HTML file:

<style>
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f5f5f5;
}
header {
    background-color: #4CAF50;
    color: white;
    padding: 1em;
    text-align: center;
}
.container {
    width: 80%;
    margin: auto;
    overflow: hidden;
}
section {
    background: white;
    padding: 2em;
    margin: 2em 0;
}
</style>

This CSS code styles the body, header, container, and section elements. You can customize these styles to fit your design needs.
