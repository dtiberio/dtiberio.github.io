Just Enough HTML to Get Started with Web Scraping

Introduction

HTML (HyperText Markup Language) is the standard language for creating web pages. Understanding its structure is essential for web scraping because you'll be extracting data from HTML documents. This tutorial covers the basics of HTML that you'll need to get started with web scraping.

Basic HTML Structure

Every HTML document has a basic structure that looks like this:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>

</body>
</html>

Key parts of an HTML document:

  • <!DOCTYPE html>: Defines the document type as HTML5.
  • <html>: Root element of an HTML page.
  • <head>: Contains meta-information about the HTML document (e.g., title, charset).
  • <title>: Sets the title of the webpage (displayed in the browser tab).
  • <body>: Contains the content of the HTML document, visible to users.

Common HTML Tags

Here are some common HTML tags you'll encounter:

  • <h1> to <h6>: Headings, <h1> is the highest (most important) level, <h6> is the lowest.
  • <p>: Paragraph.
  • <a>: Anchor (link). Example: <a href="https://www.example.com">This is a link</a>.
  • <img>: Image. Example: <img src="image.jpg" alt="Description">.
  • <div>: Division or section in an HTML document.
  • <span>: Inline container for text.
  • <ul>: Unordered list.
  • <ol>: Ordered list.
  • <li>: List item.
  • <table>: Table.
  • <tr>: Table row.
  • <td>: Table cell.

HTML Attributes

HTML tags can have attributes that provide additional information about the element. Common attributes include:

  • id: Uniquely identifies an element.
  • class: Classifies an element for styling purposes.
  • src: Specifies the source of an image.
  • href: Specifies the URL of a link.
  • alt: Provides alternative text for an image.
  • title: Adds extra information about an element (displayed as a tooltip).

Example HTML Document

Here is a complete example of an HTML document:

<!DOCTYPE html>
<html>
<head>
    <title>Sample Web Page</title>
</head>
<body>

    <h1 id="main-heading">Welcome to My Web Page</h1>

    <div class="content">
        <p>This is a paragraph of text.</p>
        <p>Here is an image:</p>
        <img src="example.jpg" alt="Example Image">
    </div>

    <a href="https://www.example.com" title="Example">Visit Example.com</a>

</body>
</html>

Using HTML Knowledge for Web Scraping

When web scraping, you'll use a library like BeautifulSoup in Python to parse HTML documents and extract data. Here's a simple example:

from bs4 import BeautifulSoup
import requests

url = 'https://www.example.com'
response = requests.get(url)

soup = BeautifulSoup(response.text, 'html.parser')

# Extracting the main heading
main_heading = soup.find('h1', id='main-heading').text
print(main_heading)

# Extracting all paragraphs
paragraphs = soup.find_all('p')
for p in paragraphs:
    print(p.text)

This script fetches a webpage, parses its HTML, and extracts the main heading and all paragraphs.
